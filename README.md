# YuKoike's VPM Repository

YuKoikeのVRChat向けUnityパッケージを配信するVPMリポジトリです。

## リポジトリの追加方法

### VRChat Creator Companion (VCC)を使用

1. VRChat Creator Companion (VCC) を開く
2. **Settings** → **Packages** → **Add Repository** をクリック
3. 以下のURLを入力して追加:
   ```
   https://yukoike224.github.io/vpm-avatar-editor-tools/index.json
   ```
4. プロジェクトを選択
5. パッケージ一覧から **YuKoike Avatar Editor Tools** を選択してインストール

## パッケージ一覧

### YuKoike Avatar Editor Tools
VRChatアバター制作・改変を効率化するUnityエディタ拡張ツール集

**主な機能:**
- Copy Asset Full Path
- GameObject Component Remover
- GameObject Component Copier
- Create Work Folders
- FT Avatar Modifier
- FT Avatar Parameter Optimizer
- Skybox Changer
- Create Kaihen (マテリアルバリアント作成)

詳細: [パッケージのREADME](https://github.com/yukoike224/vpm-avatar-editor-tools#readme)
